# Project Management App

## 📌 概要
Tailwind CSS を利用し、UI/UX を意識しながら構築したシンプルなプロジェクト管理アプリです。プロジェクトの作成とタスク管理が行えます。

## 🛠️ 技術スタック
- React
- Tailwind CSS
- useRef / useImperativeHandle

## 🎯 主な機能
- プロジェクトの作成・削除
- 各プロジェクトに対するタスク追加

## 💡 学習ポイント
- Tailwind CSS による効率的なスタイリング
- モーダル制御パターンの再利用
- コンポーネント構造に応じた UI 設計
